Key Architectural Differences

Python Monolith

Single AIAgent class (~7,800 lines) with ThreadPool-based parallel tool execution. Direct LLM API calls with provider-specific adapters.

TypeScript + SDK Runtime

Uses external Pi agent SDK for the core loop. Lane-based async concurrency with WebSocket gateway. Modular compiled JS (47 subdirectories).

FTS5 Session Memory

Full-text search across all past sessions via SQLite FTS5. Keyword-based recall. Session search is a first-class tool in every conversation.

Hybrid Vector + BM25 Memory

sqlite-vec for vector search with local EmbeddingGemma 300M. 70% vector + 30% BM25 hybrid ranking. File watcher for real-time re-indexing.

RL Training Built-in

Trajectory capture, compression, batch generation, and SWE benchmark runner. Designed to self-improve via GRPO reinforcement learning.

Plugin Ecosystem

30 extension plugins, device pairing (nodes), web control UI, canvas rendering. Focus on extensibility and multi-device orchestration.
